# portfolio_polenov_dv

## Github Page

```shell
https://polenov-dv.github.io/portfolio_polenov_dv/
```

## Cloning a repository

```shell
git clone https://github.com/polenov-dv/portfolio_polenov_dv.git
```

## Install a package

```shell
npm install
```

## Usage
Run the project locally

```shell
npm run start
```
